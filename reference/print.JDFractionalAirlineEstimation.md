# Print method for 'JDFractionalAirlineEstimation' objects

This function prints informations on the result of a Fractional Airline
model (classe JDFractionalAirlineEstimation).

## Usage

``` r
# S3 method for class 'JDFractionalAirlineEstimation'
print(x, digits = max(3L, getOption("digits") - 3L), ...)
```

## Arguments

- x:

  An object of class 'JDFractionalAirlineEstimation'.

- digits:

  Number of digits to round numerical values (default is 3 or digits - 3
  from options).

## Value

The original object 'x'.
